# Eval corpus

A small **synthetic, fictional** document set used by the RAG eval suites.
Everything here describes a made-up person ("Mara Okonkwo") and her invented
projects. Fiction is deliberate: it means the golden question/answer pairs have
fully controlled answerability (we know exactly which chunk answers each
question) and there are no privacy concerns in committing the corpus to git.

Files are ordinary markdown so they exercise the same ingestion + heading-aware
chunking path as real user documents.
